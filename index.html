TravelLedger 旅遊帳簿與預算管理系統
UI/UX 設計 • Use Case/ERD • 全端程式開發

「TravelLedger」是基於 ASP.NET Core MVC (C#) 框架開發，並採用 SQL Server 作為資料庫的系統。提供完整的記帳 CRUD 功能，整合行程時間軸與實體圖片上傳至伺服器端的功能。系統內建複雜的分帳演算法與匯率轉換機制，涵蓋「行前預算編列」、「旅途中記帳與行程對照」到「行後財務結算」的完整旅遊生命週期。
